# Telegram-бот (aiogram 3)

Готовый бот с меню, пересылкой сообщений, анонимными жалобами и защитой от спама.

## Установка

```bash
pip install -r requirements.txt
```

## Настройка

1. Создай бота у [@BotFather](https://t.me/BotFather) и получи токен.
2. Открой `bot.py` и впиши токен в переменную `BOT_TOKEN`.
3. Узнай свой Telegram ID:
   - запусти бота (`python bot.py`),
   - напиши ему `/id`,
   - скопируй число в переменную `ADMIN_ID` в `bot.py`,
   - перезапусти бота.
4. (Опционально) поправь текст в `ABOUT_TEXT` под себя.

## Запуск

```bash
python bot.py
```

## Что умеет бот

| Кнопка | Поведение |
|---|---|
| 📩 Связаться со мной | Пользователь пишет сообщение → бот пересылает тебе **с указанием** имени, username и ID |
| 💡 Есть интересное предложение? | То же самое, но с другой подписью |
| ⚠️ Жалоба или просьба | Сообщение пересылается **анонимно** — без данных пользователя |
| 🕵️ Кто я такой? | Показывает текст из `ABOUT_TEXT` |
| `/id` | Показывает Telegram ID отправителя (для настройки `ADMIN_ID`) |
| `/menu` | Открыть меню заново |
| `/cancel` или «❌ Отмена» | Отменить текущее действие |

## Защита от спама

Каждый пользователь может отправить тебе одно сообщение (через любую из кнопок)
не чаще, чем раз в **60 секунд** (`SPAM_INTERVAL` в `bot.py`). При попытке
отправить чаще бот вежливо просит подождать и показывает, сколько секунд осталось.

## Бонус: ответ прямо из Telegram

Когда тебе приходит сообщение от бота (в том числе анонимная жалоба), просто
ответь на него (Reply) прямо в Telegram — бот сам доставит твой ответ
отправителю. Для анонимных жалоб личность отправителя при этом тебе всё
равно не показывается, бот лишь хранит связь "сообщение → кому ответить"
у себя в памяти.

## Важно

- `user_last_message` (антиспам) и `reply_map` (связь для ответов) хранятся
  **в памяти процесса**. При перезапуске бота они сбрасываются. Для
  продакшен-нагрузки или нескольких экземпляров бота лучше перенести их
  в Redis или БД.
- Не публикуй токен бота в открытых репозиториях — лучше хранить его в
  переменной окружения или `.env`-файле.
